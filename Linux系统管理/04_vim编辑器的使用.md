## 1. vim编辑器介绍

模式化

* 命令模式
* 插入模式
* 末行模式

使用格式

```
# vim 文件名称
```

## 2. 模式间的切换

命令模式——>插入模式

* `a`：光标右移一位
* `i`：光标不动
* `o`：光标向下另起一行
* `O`：光标向上另起一行

插入模式——>命令模式

* `Esc`

命令模式——>末行模式

* `:`

## 3. 末行模式

* `:wq`：保存退出

* `:q!`：强制退出

* `:set nu`：显示行号

* `:s`：快速替换

```
:s/linux/python		换光标后一个
:s/linux/python/g		换当前行所有
:14,20 s/linux/python/g		换14到20行所有
:% s/linux/python/g		换全文所有
```

## 4. 命令模式

## 移动光标

* `j`：下
* `k`：上
* `l`：右
* `h`：左
* `^`：移动到行首
* `$`：移动到行尾

* `G`：移动到文件最后一行
* `gg`：移动到文件第一行
* `28gg`：移动到文件第28行

## 删除

* `x`：删除单个字符
* `d$`：删除到行尾
* `d^`：删除到行首
* `dw`：删除单词
* `dG`：删除到最后一行
* `dd`：删除整行
* `3dd`：删除三行

## 替换

* `r`：替换单个字符
* `R`：进入替换模式，连续替换

## 复制、粘贴

* `y$`：复制到行尾
* `y^`：复制到行首

* `yy`：复制整行
* `3yy`：复制三行
* `yw`：复制单词
* `p`：粘贴
* `30pp`：粘贴30遍

## 剪切、粘贴

* 删除和粘贴组合

## 搜索

* `/text`：搜索text
  * `n`：向下查找
  * `N`：向上查找

## 撤销

* `u`：撤回
