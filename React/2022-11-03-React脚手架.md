## 使用create-react-app创建react应用📚

1. 全局安装

```shell
npm i create-react-app -g
```

2. 创建

```shell
create-react-app <project_name>
```

3. 进入项目文件夹

```shell
cd <project_name>
```

4. 启动项目

```shell
npm start
```
