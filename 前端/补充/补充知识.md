

### 数组和字符串


|         |       数组       |   字符串   |
| :-----: | :--------------: | :--------: |
|  slice  |     截取数组     | 截取字符串 |
| splice  | 删除、添加、替换 |            |
| replace |                  |    替换    |
|  split  |                  | 分隔为数组 |
|  join   |   分割为字符串   |            |

### this

this指向函数的拥有者

回调函数的this指向window

### overflow

`hidden` 超出父元素的部分隐藏 应用：解决塌陷问题

`scrool` 超出父元素的部分以滚动条显示

`overflow-x: hidden` x方向隐藏 y方向以滚动条显示

`overflow-y: hidden` y方向隐藏 x方向以滚动条显示
