### 二、安装MySQL 5.7 

#### 1、配置MySQL仓库 

```
[root@localhost ~]# cat /etc/yum.repos.d/mysql.repo
[mysql57]
name=mysql57
baseurl=ftp://10.10.11.100/software/mysql/
enabled=1
gpgcheck=0
```



#### 2、安装MySQL服务端软件  

```
[root@localhost ~]# yum install -y mysql-community-server 
```



#### 3、启动mysql服务  

```
[root@localhost ~]# systemctl start mysqld
[root@localhost ~]# systemctl enable mysqld
[root@localhost ~]# 
[root@localhost ~]# ps -elf | grep mysql
1 S mysql      1900      1  3  80   0 - 296188 poll_s 11:10 ?       00:00:00 /usr/sbin/mysqld --daemonize --pid-file=/var/run/mysqld/mysqld.pid
0 S root       1949   1069  0  80   0 - 28165 pipe_w 11:10 pts/0    00:00:00 grep --color=auto mysql

[root@localhost ~]# netstat -antp | grep mysql
tcp6       0      0 :::3306                 :::*                    LISTEN      1900/mysqld       
```

### 三、MySQL相关文件 

```
1、配置文件 --- /etc/my.cnf  

[mysqld]
datadir=/var/lib/mysql					//数据目录 
socket=/var/lib/mysql/mysql.sock		//定义套接字文件存储位置，套接字文件(IP;port)， 用于接收客户端连接请求 

log-error=/var/log/mysqld.log			//定义错误日志
pid-file=/var/run/mysqld/mysqld.pid		//定义pid文件 


2、默认数据目录 
	/var/lib/mysql  
	
3、错误日志 
	/var/log/mysqld.log 
```
