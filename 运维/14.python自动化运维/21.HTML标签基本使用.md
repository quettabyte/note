### 三、表单标签

```
<form action="url地址" method="get/post">
	表单元素
	表单元素
</form>

	action=		数据提交地址
	method=		数据提交方法
```

##### 提交数据的方法

```
1、get方法

	以字符串拼接的方式进行提交 
	http://x.x.x.x/?a=10&b=20&c=30
	
	request.GET接收get方法提交的数据 {"a":10, "b":20, "c":30}
	
2、post方法
	
	将表单中的数据封装HTTP报文中进行提交，在浏览器中按F12可查看 
	
	request.POST接收表单通过post方法提交的数据	{"user": "tome"}
```



#### 1、文本框

```
    <div>
        <span>用户名： </span>
        <input type="text" name="user"/>
    </div>

	type="text"		文本框
	name="xxxx"		便于后台接收数据
```

#### 2、提交按钮

```
<input type="submit" value="确认" />
```

#### 3、密码框 

```
        <div>
            <span>密码： </span>
            <input type="password" name="pwd"/>
        </div>
```

#### 4、单选按钮

```
		<div>
            <span>性别: </span>
            <input type="radio" name="sex" value="1"/>
            <span>男</span>
            <input type="radio" name="sex" value="2"/>
            <span>女</span>
        </div>
        
        注意:
        	1、name=	名称要一致
        	2、单选按钮提交的是value指定的数值
```
