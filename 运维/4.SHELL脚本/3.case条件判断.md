## case条件判断

### 一、case条件判断语法 

```
适用于某个变量拥有多种固定的不同的取值
```

```
语法: 

case 变量 in
值1)
   执行的操作
   ;;
值2)
   执行的操作
   ;;
值3)
   执行的操作
   ;;
*)
   执行的操作
   ;;
esac
```

#### 案例: 判断用户输入 

```
#!/bin/bash
#

read -p "数据: " data

case $data in
linux)
   echo "redhat"
   ;;
windows)
   echo "microsoft"
   ;;
*)
   echo "Other"
   ;;
esac
```

```
#!/bin/bash
#

read -p "数据: " data

case $data in
linux|Linux)							// | 表示或者关系 
   echo "redhat"
   ;;
windows|Windows)
   echo "microsoft"
   ;;
*)
   echo "Other"
   ;;
esac
```



### 二、位置变量 

```
$1, $2, $3......$9, ${10}
	$1：代表命令的第一个参数
	$2：代表命令的第二个参数
	
$0： 命令本身 

$#: 参数的个数
```

```
#!/bin/bash
#

case $1 in
  linux|Linux)
      echo "Redhat"
      ;;
  windows|Windows)
      echo "Microsoft"
      ;;
  *)
      echo "Other"
      ;;
esac
```



```
#!/bin/bash
#


# 判断是否有参数

if [ $# -eq 0 ]; then
   echo "帮助: $0 {linux|windows|other}"
   exit 78
fi

case $1 in
  linux|Linux)
      echo "Redhat"
      ;;
  windows|Windows)
      echo "Microsoft"
      ;;
  *)
      echo "Other"
      ;;
esac
```
